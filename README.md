# Trabalho-N1

Este projeto foi desenvolvido em linguagem C para resolver equações do 1º e 2º grau.

## Funcionalidades

- Menu principal
- Resolução de equação do 1º grau
- Resolução de equação do 2º grau
- Exibição do passo a passo dos cálculos
- Opção "Sobre o Programa"
